# FizzBuzz demo

FizzBuzz-demo is an express application for getting fizzbuzz value per input number 

## Installation

for local development:

    1. clone the project    
    2. npm install
    3. npm run
    for running tests:
    npm test
    
for local development:

    1. clone the project    
    2. npm install
    3. npm run
    for running tests:
    npm test
    
for building docker image:

    1. clone the project    
    2. docker build -t TARGET .
